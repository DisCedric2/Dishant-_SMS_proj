# Django Student Management System

A full-stack web application built with **Django, MySQL, HTML, CSS, and JavaScript** to manage student records efficiently. Developed as part of a technical assessment.

---

## Features

- **Add/Edit/Delete students** with full CRUD functionality  
- **Client-side and server-side validation** including duplicate email check and valid age  
- **AJAX integration** for deleting students without page reload  
- **Filtered list:** Shows only students aged 18+  
- **Highlighting:** Students enrolled in the same course are visually highlighted  
- **Success/Error messages** using Django messages framework  

---

## Tech Stack

- Backend: Django, MySQL  
- Frontend: HTML, CSS, JavaScript, Django Templates  
- AJAX for real-time actions  

---

## Setup & Run

1. Clone the repository  
2. Create and activate a virtual environment  
3. Install dependencies: `pip install -r requirements.txt`  
4. Configure MySQL database in `settings.py`  
5. Run migrations: `python manage.py migrate`  
6. Start server: `python manage.py runserver`  
7. Open [http://127.0.0.1:8000/](http://127.0.0.1:8000/) in your browser
